Congreso Nacional de Sistemas (CONASI 2018)

Por Osvaldo, 2018-11-07 23:00

En el mes de noviembre del 2018 participé en el Congreso Nacional de Sistemas (CONASI 2018) organizado por el Tecnológico Nacional de México Campus Instituto Tecnológico de Matehuala, en Matehuala, San Luis Potosí, con una conferencia y un taller.

La conferencia

El 7 de noviembre impartí la conferencia El software libre como aliado de la legalidad en tu empresa

Se explicó la diferencia entre Software Libre y el software propietario, así como las libertades que se gozan con el uso del software libre, las diferentes licencias y un listado de software que pueden usar los futuros emprendedores. También se mostraron ejemplos.


El taller

El 8 de noviembre impartí el taller “Compartiendo código con Git y Github”, aquí creamos una cuenta en Github y creamos un repositorio en el cual trabajamos, después de conocer los comandos básicos de Git, con un proyecto en equipo; se les recordó un diagrama como el siguiente:


Es de notar, además de la belleza natural con que cuenta Matehuala y sus alrededores, las enormes ganas de aprender así como la unión y compañerismo entre alumnos y maestros :-D
